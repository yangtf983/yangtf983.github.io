1 Feature Exploitation

我们讲过的运用特征转换的第一种方法是 kernel . 从支持向量机的对偶问题中我们引出了 kernel 的概念，能够使用 kernel 的模型通过改变 kernel 可以做到非常多不一样的事情，下图罗列了我们讲过的 kernel 和能够使用 kernel 的模型，有一些模型我们没有说明可以加入 kernel，但是其中有内积，例如主成分分析（PCA）、k 近邻算法，都可以推广到有 kernel 的版本。

我们讲过的运用特征转换的第二种方法是聚合。也就是把 gtg_tgt​ 当作特征转换，把转换后的数据或一致、或线性、或条件地结合起来就产生了我们的三类聚合模型。下图中总结了我们常用的基本算法 gtg_tgt​ 和聚合算法。

在我们介绍的模型中还有一个共同的但是常常被忽略的特征转换，就是把数据压缩/投影到低维空间上。例如：decision stump 可以被看作是把数据压缩到一维考虑怎么分割；随机森林中我们讲过一种增加随机性的方法就是对数据随机进行一次低维度的投影；自动编码器和主成分分析也是在找一个低维度投影；矩阵分解是在把一个很高维的稀疏的向量投影到一个低维的稠密的向量。这种数据降维的方法在实际问题中非常常用。
